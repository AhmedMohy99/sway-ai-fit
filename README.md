# Sway AI Fit

A Vite + React site with a dark luxury streetwear aesthetic inspired by swaymaverick.com, including interactive 3D hoodie and sweatpants models sized to realistic apparel dimensions.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push repository to GitHub.
2. Import into Vercel.
3. Build command: `npm run build`
4. Output directory: `dist`
